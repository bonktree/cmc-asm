Microsoft Macro Assembler
==============
* Для установки нужно запустить setup_unix.sh. По умолчанию
  устанавливаются wine и dosbox для компиляции и запуска программ,
  а также редактор Geany.

* В файле schem.asm находится каркас программы на языке Ассемблера с
  комментариями. Для написания своих программ сохраните этот файл под
  другим именем.

* Для запуска своей программы в Geany нужно нажать F8.

* Для перехода на русский язык и обратно во время выполнения программы
  нужно нажать правый Ctrl.

* При работе в другом редакторе, а также из командной строки нужно
  вызвать fullma.sh с аргументом-именем файла с программой, например,
  ./masm/fullma.sh prog.asm
